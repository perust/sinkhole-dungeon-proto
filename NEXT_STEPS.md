- [ ] 엔딩 후보 3: 플레이어가 들고 나온 유물 때문에 싱크홀이 더 깊어지고 있었다

### 2. 아이템 Heat/태그 고도화

현재는 등급 기반 Heat에 가깝다. 다음에는 아이템 개별 속성으로 분리해야 함.

- [ ] `heat`: 암시장 판매 시 의심도 상승량
- [ ] `noise`: 집는 순간 추적도 상승량
- [ ] `marked`: 위원회가 추적 중인 회수물
- [ ] `truth`: 진실 조각 여부
- [ ] `family`: 실종자 가족에게 돌려줄 수 있는 유품
- [ ] `fragile`: 버리고 도망 시 파손되어 일부 보상만 남음

## 재미를 더 올리는 후속 시스템

### 3. 방 타입 확장

MVP 3종은 구현됨. 다음은 반복감이 더 필요할 때 추가한다.

- [ ] 무너진 통로: 조명 소모 증가, 대신 고가 회수물 확률 증가
- [ ] 위원회 감시소: 의심도가 높으면 불리한 이벤트
- [ ] 실종자 흔적방: 유품/가족 루트 연결

### 4. 출구 선택

- [ ] 공식 출구: 안전하지만 검문 있음
- [ ] 균열 출구: 검문 회피, 대신 부상/아이템 손상 가능
- [ ] 암시장 통로: 비용 지불, 불법 유물 보존 가능

### 5. 생존자 시스템

- [ ] 정비공 생존자: 장비 강화 비용 감소
- [ ] 의사 생존자: 왜곡/실패 패널티 완화

- [ ] 구출하지 않은 생존자가 후속 런에서 변이체/뉴스/평판 패널티로 돌아오는 구조
