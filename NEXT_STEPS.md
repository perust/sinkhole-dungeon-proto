## 바로 해야 할 것 — 우선순위 높음

### 1. 아이템 Heat/태그 고도화

현재는 등급 기반 Heat에 가깝다. 다음에는 아이템 개별 속성으로 분리해야 함. 방 선택 3지선다의 전략성을 살리려면 엔딩보다 먼저 이 작업이 필요하다.

**v1 완료 (heat/noise/fragile):** 6종 물건에 `heat`·`noise`·`fragile` 태그 추가, `itemHeat`/`itemNoise`/`itemFragile` 헬퍼로 접근하며 태그 누락 시 등급 기준으로 보정. 나머지 `marked`/`family`/`truth` 플래그와 6조각 엔딩은 다음 회차로 유지.

- [x] `heat`: 암시장 판매 시 의심도 상승량 (물건별 합산: 배터리 3·배관 2·데이터칩 9·노트 7·코어 15·유물 20)
- [x] `noise`: 집는 순간 추적 압박 (low/medium/high, 조심/재빨리와 조합)
- [ ] `marked`: 위원회가 추적 중인 회수물
- [ ] `truth`: 진실 조각 여부 (현재는 문자열 필드만 존재, 플래그화 보류)
- [ ] `family`: 실종자 가족에게 돌려줄 수 있는 유품
- [x] `fragile`: 버리고 도망 시 파손 — 결과 문구에 `깨지는 소리가 났다` 노출 (데이터칩·노트·유물)

### 2. 진실 조각 6개 엔딩 — v1 완료

**v1 완료:** 암시장 판매로 마지막(6/6) 진실 조각이 처음 채워지면 강화 화면 대신 엔딩 화면을 띄운다. RP·의심도·의뢰·저장은 그대로 반영한 뒤 전환하며, 매 런 반복 발동하지 않는다. `endingSeen` 저장 플래그(구버전 기본 false, 하위호환이라 SAVE_VERSION 유지)로 확인 여부를 기록하고, 시작 화면 다음 목표는 확인 전 `진실 확인하기` → 확인 후 `심층 루프 계속`으로 바뀐다.

- [x] 진실 조각 6개 수집 시 엔딩 화면 표시
- [x] 엔딩 리빌: 위원회의 은폐/재봉인, 직인 일치, 내가 들고 나온 물건 때문에 아래가 한 층 더 열림
- [x] 엔딩 후 `심층 루프 계속`(강화 루프 유지) / `기록 초기화` 선택 제공
- [x] `심층 루프 계속` 시 획득 RP가 반영된 강화 화면으로 이어짐
