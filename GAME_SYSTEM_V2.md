# Sinkhole Dungeon — 게임 시스템 고도화 v2

## 현재 방향

**장르:** 모바일 추출형 던전 루프 / Extraction Roguelite  
**핵심 문장:** “딱 한 층만 더 — 그걸 들고 나올 수 있을까?”

플레이어는 도심 싱크홀 아래의 왜곡 던전에 들어가 회수물을 챙기고, 회수자가 따라잡기 전에 지상으로 돌아온다. 지상에서는 회수물을 **위원회**에 넘길지, **암시장**에 팔지 선택한다. 이 선택으로 RP, 의심도, 진실 조각, 다음 목표가 달라진다.

---

## 이번 프로토타입에 반영된 시스템

### 1. 추출 루프

1. 내려가기
2. 층마다 5~7개 노드짜리 작은 맵 생성
3. 현재 위치의 출구 기반 갈림길 선택
4. 노드 도착: 회수물 / 몬스터 이벤트 / 계단 여부 확인
5. 회수물을 줍거나, 기다리거나, 다른 길로 이동
6. 줍는 순간 추적 시작, 추적도 상승
7. 계단 아래로 / 버리고 도망 / 들고 나가기 선택
8. 지상 판매처 선택
9. RP 획득, 의심도 변화, 진실 조각 획득
10. 강화 후 재진입

### 2. 추적도

던전 내부의 핵심 압박 자원.

- 회수물을 집으면 추적 시작
- 시간이 지나면 추적도 상승
- 더 깊이 내려가면 조명과 위험이 악화
- 위험이 100%가 되면 실패
- “버리고 도망”으로 가장 비싼 짐을 미끼로 던져 위험을 낮춤

### 3. 판매처 선택

#### 위원회

- 낮은 판매가
- 의심도 감소
- 의뢰 조건에 따라 보너스 가능
- 안전하지만 진실 조각 획득은 제한적

#### 암시장

- 높은 판매가
- 의심도 상승
- 회수물의 진실 조각 획득 가능
- 돈은 빠르게 벌지만 장기적으로 검문/위험이 커질 수 있음

### 4. 의심도

지상 쪽 장기 압박 자원.

- 암시장 판매 시 증가
- 위원회 제출 시 감소
- 실패/검문/뉴스/향후 콘텐츠와 연결 예정

### 5. 진실 조각

미스터리 해금 장치.

- 암시장에 팔면 회수물의 숨은 진실이 하나씩 열린다
- 현재는 6개 회수물 각각이 진실 조각을 가진 구조
- 향후 엔딩/원점실/위원회 반전과 연결 예정

### 6. 오늘의 의뢰

반복 진입의 목적을 만드는 장치.

- 합법 샘플 반납
- 뜨거운 증거
- 하층 동선 확인

의뢰 완료 시 RP 보너스와 의심도 변화가 적용된다.

### 7. UI/UX 개선

- 시작 화면에 의심도, 진실 조각, 오늘의 의뢰, 다음 목표 표시
- 던전 화면에 추적도/조명/깊이 레일 표시
- 판매처 선택 화면 분리
- 귀환 화면에 판매처, 런 요약, 진실 조각, 거리 뉴스, 다음 목표 표시
- 기존 도트 에셋을 HUD/스테이지/아이템/캐릭터에 유지 적용

### 8. 저장/이어하기

런을 넘는 진행도는 `localStorage`에 자동 저장한다.

- 저장 대상: RP, 가방/조명/무기 강화 레벨, 최고 깊이, 총 획득 RP, 의심도, 진실 조각, 오늘의 의뢰 진행
- 저장 데이터 버전 필드 포함
- 깨진 저장값이나 버전이 맞지 않는 저장값은 안전하게 무시
- 시작 화면에서 `기록 초기화` 가능

### 9. 작은 맵 / 동적 갈림길

층에 들어가면 고정 3지선다가 아니라 5~7개 노드짜리 작은 맵이 생성된다.

- 각 노드는 출구, 방 유형, 회수물, 몬스터 이벤트를 가질 수 있다.
- 선택지는 현재 노드의 출구에 따라 2~4개로 달라진다.
- 예: 왼쪽 문, 정면 복도, 낮은 창고, 관리실, 환풍구, 무너진 통로, 오른쪽 균열, 계단 아래로.
- 막다른 곳은 짧게 표시하고, 계단은 더 깊은 층으로 이어진다.

목표는 플레이어가 “맵 안을 실제로 이동한다”는 감각을 갖게 하는 것이다.

### 10. 몬스터 시야 이벤트

회수자는 단순 게이지로만 다가오지 않고, 특정 노드/출구에서 보인다.

- 직선 끝에 보임: 위험 상승, 해당 출구에 `회수자가 보인다` 표시.
- 갈림길을 지나감: `기다리기`로 흘려보낼 수 있고, 움직이면 들킨다.
- 움직이면 들킴: 다음 이동 시 위험 급증/추격 시작, `기다리기`로 완화.

`기다리기`는 상황이 있을 때만 동적으로 나온다.

## MVP 판단 기준

이 버전에서 검증해야 하는 질문은 하나다.

> “플레이어가 한 번 더 내려가고 싶은가?”

구체적으로는:

- 비싼 회수물을 들고 나올 때 긴장감이 있는가?
- 위원회와 암시장 선택이 숫자 이상의 딜레마로 느껴지는가?
- 진실 조각 때문에 반복 동기가 생기는가?
- 강화 선택이 다음 런의 욕심을 키우는가?
- 모바일 한 화면에서 판단이 과하게 피곤하지 않은가?

## 다음 설계 방향

1. **Heat/태그**를 아이템별 명시 속성으로 확장하기
2. **진실 조각 6개 엔딩** 추가하기
3. **방 타입 확장**: 무너진 통로, 위원회 감시소, 실종자 흔적방
4. **액션 피드백**: 위험 50/80/95% 연출, 사운드, 흔들림, 회수자 접근감 강화
5. **도트 리소스 분리**: 현재 스트립/시트를 개별 스프라이트로 잘라 가독성 개선
