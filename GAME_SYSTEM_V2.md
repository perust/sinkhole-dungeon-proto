# Sinkhole Dungeon — 게임 시스템 고도화 v2

## 현재 방향

**장르:** 모바일 추출형 던전 루프 / Extraction Roguelite  
**핵심 문장:** “딱 한 층만 더 — 그걸 들고 나올 수 있을까?”

플레이어는 도심 싱크홀 아래의 왜곡 던전에 들어가 회수물을 챙기고, 회수자가 따라잡기 전에 지상으로 돌아온다. 지상에서는 회수물을 **위원회**에 넘길지, **암시장**에 팔지 선택한다. 이 선택으로 RP, 의심도, 진실 조각, 다음 목표가 달라진다.

---

## 이번 프로토타입에 반영된 시스템

### 1. 추출 루프

1. 내려가기
2. 방 선택: 안전 캠프 / 폐쇄 연구실 / 회수자 둥지
3. 회수물 확인
4. 챙기기
5. 추적도 상승
6. 더 깊이 / 버리고 도망 / 들고 나가기 선택
7. 지상 판매처 선택
8. RP 획득, 의심도 변화, 진실 조각 획득
9. 강화 후 재진입

### 2. 추적도

던전 내부의 핵심 압박 자원.

- 회수물을 집으면 추적 시작
- 시간이 지나면 추적도 상승
- 더 깊이 내려가면 조명과 위험이 악화
- 위험이 100%가 되면 실패
- “버리고 도망”으로 가장 비싼 짐을 미끼로 던져 위험을 낮춤

### 3. 판매처 선택

#### 위원회

- 낮은 판매가
- 의심도 감소
- 의뢰 조건에 따라 보너스 가능
- 안전하지만 진실 조각 획득은 제한적

#### 암시장

- 높은 판매가
- 의심도 상승
- 회수물의 진실 조각 획득 가능
- 돈은 빠르게 벌지만 장기적으로 검문/위험이 커질 수 있음

### 4. 의심도

지상 쪽 장기 압박 자원.

- 암시장 판매 시 증가
- 위원회 제출 시 감소
- 실패/검문/뉴스/향후 콘텐츠와 연결 예정

### 5. 진실 조각

미스터리 해금 장치.

- 암시장에 팔면 회수물의 숨은 진실이 하나씩 열린다
- 현재는 6개 회수물 각각이 진실 조각을 가진 구조
- 향후 엔딩/원점실/위원회 반전과 연결 예정

### 6. 오늘의 의뢰

반복 진입의 목적을 만드는 장치.

- 합법 샘플 반납
- 뜨거운 증거
- 하층 동선 확인

의뢰 완료 시 RP 보너스와 의심도 변화가 적용된다.

### 7. UI/UX 개선

- 시작 화면에 의심도, 진실 조각, 오늘의 의뢰, 다음 목표 표시
- 던전 화면에 추적도/조명/깊이 레일 표시
- 판매처 선택 화면 분리
- 귀환 화면에 판매처, 런 요약, 진실 조각, 거리 뉴스, 다음 목표 표시
- 기존 도트 에셋을 HUD/스테이지/아이템/캐릭터에 유지 적용

### 8. 저장/이어하기

런을 넘는 진행도는 `localStorage`에 자동 저장한다.

- 저장 대상: RP, 가방/조명/무기 강화 레벨, 최고 깊이, 총 획득 RP, 의심도, 진실 조각, 오늘의 의뢰 진행
- 저장 데이터 버전 필드 포함
- 깨진 저장값이나 버전이 맞지 않는 저장값은 안전하게 무시
- 시작 화면에서 `기록 초기화` 가능

### 9. 방 선택 3지선다

층에 들어가면 바로 회수물이 뜨지 않고, 먼저 방을 고른다.

- 안전 캠프: 조명 회복, 낮은 위험
- 폐쇄 연구실: 희귀/진실 단서 쪽 확률 상승, 약간의 비용
- 회수자 둥지: 고가 회수물 확률 상승, 추적도 압박 증가

목표는 매 런마다 “어디를 털고 어디를 포기할지”를 작게 판단하게 만드는 것이다.

## MVP 판단 기준

이 버전에서 검증해야 하는 질문은 하나다.

> “플레이어가 한 번 더 내려가고 싶은가?”

구체적으로는:

- 비싼 회수물을 들고 나올 때 긴장감이 있는가?
- 위원회와 암시장 선택이 숫자 이상의 딜레마로 느껴지는가?
- 진실 조각 때문에 반복 동기가 생기는가?
- 강화 선택이 다음 런의 욕심을 키우는가?
- 모바일 한 화면에서 판단이 과하게 피곤하지 않은가?
