# 도심 싱크홀 — 회수 잠수 프로토타입

가볍고 직관적인 던전 회수(extraction) 게임의 첫 플레이어블 프로토타입입니다.

## 핵심 훅

> 던전이 내가 훔친 물건을 다시 가져가려 한다.

플레이어는 도심 싱크홀 아래 던전에 들어가 회수물을 훔치고, 회수자가 물건을 되찾으러 쫓아오기 전에 지상으로 귀환해 판매합니다. 얻은 RP로 가방/조명/무기 중 하나를 강화하고 다시 더 깊이 내려갑니다.

## 구현된 루프

1. `입장`으로 던전 1층 진입
2. 방 선택: `안전 캠프` / `폐쇄 연구실` / `회수자 둥지`
3. `회수물 집기`로 가방에 회수물 추가
4. 회수자 추격 시작, 추적도 상승
5. `더 깊이` / `버리고 도망` / `들고 귀환` 판단
6. 귀환 성공 시 판매처 선택: `위원회` 또는 `암시장`
7. 판매 결과로 RP, 의심도, 진실 조각, 의뢰 결과가 변화
8. 가방/조명/무기 중 하나 강화
9. `다시 내려가기`로 다음 런 시작
10. 추적도가 100%가 되면 회수자가 가방을 되찾아가고 런 실패, 일부 보상만 유지

## v2 고도화 반영

- 판매처 선택: 위원회는 안전/저가/의심도 감소, 암시장은 고가/의심도 상승/진실 조각 해금
- 지상 의심도: 장기 압박 자원
- 진실 조각: 회수물별 미스터리 해금
- 오늘의 의뢰: 반복 진입의 단기 목표
- 다음 목표/거리 뉴스/런 요약: 플레이어가 왜 다시 들어가야 하는지 명확화
- 저장/이어하기: RP, 강화, 최고 깊이, 의심도, 진실 조각, 오늘의 의뢰 진행 자동 저장
- 방 선택 3지선다: 안전 캠프 / 폐쇄 연구실 / 회수자 둥지

상세 설계는 `GAME_SYSTEM_V2.md`, 남은 작업은 `NEXT_STEPS.md`를 참고합니다.


## 실행 방법

빌드 과정이 없는 정적 HTML 프로토타입입니다.

```bash
cd /opt/data/repos/sinkhole-dungeon-proto
python3 -m http.server 8125 --bind 127.0.0.1
```

브라우저에서 열기:

```text
http://127.0.0.1:8125/
```

## 파일 구조

- `index.html` — 화면 구조
- `styles.css` — 모바일 세로 HUD/UI 스타일
- `src/game.js` — 게임 상태, 회수/추격/귀환/판매/강화 루프
- `assets/pixel/` — 생성형 도트 캐릭터, 몬스터, 아이템, UI, 배경 에셋
