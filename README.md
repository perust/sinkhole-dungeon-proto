# 도심 싱크홀 — 회수 잠수 프로토타입

가볍고 직관적인 던전 회수(extraction) 게임의 첫 플레이어블 프로토타입입니다.

## 핵심 훅

> 던전이 내가 훔친 물건을 다시 가져가려 한다.

플레이어는 도심 싱크홀 아래 던전에 들어가 회수물을 훔치고, 회수자가 물건을 되찾으러 쫓아오기 전에 지상으로 귀환해 판매합니다. 얻은 RP로 가방/조명/무기 중 하나를 강화하고 다시 더 깊이 내려갑니다.

## 구현된 루프

1. `입장`으로 던전 1층 진입
2. `회수물 집기`로 가방에 회수물 추가
3. 회수자 추격 시작, 위험 게이지 상승
4. `버리고 도망` 또는 `들고 귀환` 판단
5. 귀환 성공 시 회수물 판매로 RP 획득
6. 가방/조명/무기 중 하나 강화
7. `한 번 더 내려가기`로 다음 런 시작
8. 위험이 100%가 되면 회수자가 가방을 되찾아가고 런 실패, 일부 보상만 유지

## 실행 방법

빌드 과정이 없는 정적 HTML 프로토타입입니다.

```bash
cd /opt/data/sinkhole-dungeon-proto
python3 -m http.server 8010 --bind 127.0.0.1
```

브라우저에서 열기:

```text
http://127.0.0.1:8010/
```

## 파일 구조

- `index.html` — 화면 구조
- `styles.css` — 모바일 세로 HUD/UI 스타일
- `src/game.js` — 게임 상태, 회수/추격/귀환/판매/강화 루프

## 현재 범위

- 던전 3층
- 회수자 1종
- 회수물 6종
- 강화 3종: 가방 / 조명 / 무기
- 스토리 설명은 최소화하고 핵심 루프 체감에 집중
